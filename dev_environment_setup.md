# Local Development Setup Checklist

Follow these steps to configure a local development environment with Vite so that you can run `npm run dev` to preview the project.

- [x] Create `package.json` with Vite configuration.
- [x] Create `.gitignore` to exclude `node_modules` and `dist`.
- [x] Run `npm install` to download Vite dependencies.
- [ ] Run `npm run dev` to start the local preview server.
- [ ] Verify that the website loads successfully in the browser at the local address.
